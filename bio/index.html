MOVIE NIGHT

2020



this piece was inspired by a commission done earlier in the same year as a virtual flyer for a party that included using a flashing effect in a .gif to simulate lightening.

it was done using photoshop on an ipad in a 3 hour period from start to finish.

i think this piece shows some of my technical ability in character design, color layout, and scene lighting. the linework is clean and the story is easy to read. proof of this is in the fact that the character’s emotions are clear, they’re posed dynamically, each object drawn is important to the narrative, the colors chosen are bold yet natural, and the lighting informs the mood.
